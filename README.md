# DevOps Formation

DevOps training materials, exercises, and configuration files for learning modern infrastructure practices.

## Overview

A comprehensive collection of DevOps training resources covering containerization, CI/CD pipelines, infrastructure as code, and cloud deployment strategies.

## Topics Covered

- **Docker** — Containerization fundamentals
- **Kubernetes** — Container orchestration
- **CI/CD** — Continuous Integration & Deployment
- **Terraform** — Infrastructure as Code
- **Linux** — System administration
- **Monitoring** — Observability & alerting

## Tech Stack

- **Docker** — Container runtime
- **Kubernetes** — Orchestration
- **Terraform** — IaC
- **GitHub Actions** — CI/CD
- **Linux Bash** — Scripting

## Getting Started

```bash
git clone https://github.com/skyson-ai/Devops-formation.git
cd Devops-formation
```

Follow the directory structure to navigate through different modules and exercises.
